# Smart-Contracts for Decentralized Commodity Exchange
## Quickstart

```sh
git clone git@github.com:gennadiymarin/DCE.git
cd contracts
yarn
```

## Usage
### Compile contracts:
```sh
yarn hardhat compile
```
### Deploy contracts:
```sh
yarn hardhat deploy
```
### Run tests:
```sh
yarn hardhat test
```
### Check tests coverage:
```sh
yarn hardhat coverage
```
